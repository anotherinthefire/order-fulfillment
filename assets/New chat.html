To check if the stock already exists before inserting it into the stock table, you can perform a query to check for a matching record. Here's an updated version of the code that includes the check:

php
<?php
include('../../config.php');

if ($_SERVER["REQUEST_METHOD"] == "POST") {
  // Get the submitted values
  $category_id = $_POST["category_id"];
  $prod_id = $_POST["prod_id"];
  $color_id = $_POST["color_id"];
  $size_id = $_POST["size_id"];
  $quantity = $_POST["quantity"];

  // Check if the stock already exists
  $checkQuery = "SELECT * FROM stock 
                 WHERE category_id = '$category_id' 
                 AND prod_id = '$prod_id' 
                 AND color_id = '$color_id' 
                 AND size_id = '$size_id'";

  $checkResult = $conn->query($checkQuery);

  if ($checkResult->num_rows > 0) {
    // Stock already exists, update the quantity
    $row = $checkResult->fetch_assoc();
    $existingQuantity = $row["stock"];
    $newQuantity = $existingQuantity + $quantity;

    $updateQuery = "UPDATE stock 
                    SET stock = '$newQuantity' 
                    WHERE stock_id = " . $row["stock_id"];

    if ($conn->query($updateQuery) === TRUE) {
      echo "Stock updated successfully";
    } else {
      echo "Error updating stock: " . $conn->error;
    }
  } else {
    // Stock doesn't exist, insert new stock
    $insertQuery = "INSERT INTO stock (category_id, prod_id, color_id, size_id, stock)
                    VALUES ('$category_id', '$prod_id', '$color_id', '$size_id', '$quantity')";

    if ($conn->query($insertQuery) === TRUE) {
      echo "Stock added successfully";
    } else {
      echo "Error adding stock: " . $conn->error;
    }
  }
}

// Close the database connection
$conn->close();
?>

In the updated code, after getting the submitted values, we perform a SELECT query to check if a stock record with the same category, product, color, and size already exists in the stock table. If a matching record is found, we update the quantity by adding the new quantity to the existing quantity. If no matching record is found, we proceed to insert a new stock record.

Note that this code assumes you have a primary key column named stock_id in your stock table. Adjust the code accordingly if your primary key column has a different name.

function addCategory() {
  var categoryInput = document.getElementById('categoryInput');
  var category = categoryInput.value.trim();

  if (category !== '') {
    // Send an AJAX request to the PHP script
    var xhr = new XMLHttpRequest();
    xhr.open('POST', 'add_category.php', true);
    xhr.setRequestHeader('Content-type', 'application/x-www-form-urlencoded');
    xhr.onreadystatechange = function() {
      if (xhr.readyState === XMLHttpRequest.DONE && xhr.status === 200) {
        // Handle the response from the PHP script
        var response = xhr.responseText;
        if (response === 'success') {
          alert('Category added successfully');
          // Reload the page or perform any other action
          location.reload();
        } else {
          alert('Error adding category');
        }
      }
    };
    xhr.send('category=' + encodeURIComponent(category));
  }
}
